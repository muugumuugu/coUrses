# Pluralization
 + generally using suffixes
   + /z/ or /s/
 + Dog*s*, Babi*es*
 + exceptions
  + relics of old english
  +  children,teeth,..

# Pattern - Plural Markers (sound)
+ /p/,/t/,/k/  --> /s/
+ else --> /z/
