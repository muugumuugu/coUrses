===== imp prop of open intervals =====

=== property ===
* a<b∈ ℝ :∀ x ∈ (a,b),∃ r<0,
* ∋(x-r,x+r) ⊆ (a,b)

=== proof ===
* for any point within the interval there ∃ a non empty open interval arounnd it within the parent interval .
* given (x ∈ (a,b) )<br> set (r= min {b-x,x-a}>0)
* claim ((x-r.x+r) ⊆(a,b))
* let  y ∈ (x-r,x+r) ∋ x-r<y<x+r <br> since  r ≤ b-x, → x+r ≤ b<br> similarly  r ≤ x-a, → x-r ≥ a<br> therefore (x-r,x+r) ⊆ (a,b)

=== complete extension to unions ===
* taking any **(countable or uncountable)** collection {(a_{α}, b_{α}):α ∈ Ω} of open intervals indexed by α.
* claim that if x ∈ ⋃_{α ∈ Ω}(a_{α}, b_{α}), then  ∃ r >0 ∋ (x-r,x+r)⊆ ⋃{α ∈ Ω}(a_{α}, b_{α})
* proof
	* let (x ∈ ⋃_{α ∈ Ω≥}{(a_{α},b_{α})})(→ ∃ α_0 ∈  Ω≥ ∋ x ∈ (a_{α_0},b_{α_0}))
	* therefore ∃ r >0 ∋ (x-r,x+r) ⊆ (a_{α₀},b_{α₀})⊆  ⋃{α ∈ Ω}(a_{α}, b_{α})

=== extension to finite intersections ===
* taking any **finite** collection {(a _{i}, b _{i}): i = //1,2,3// … n} of open intervals indexed by //i//.
* let x ∈ ^{n}⋂_{i=1}(a_{i},b_{i}) → ∀ i=//1,2,3// … n, x ∈(a_{i},b_{i})
* ∴ ∃ r_i<0: i=//1,2,3,//… n} ∋ (x-r_i,x+r_i)⊆(a_{i},b_{i})
* set r=𝑚𝑖𝑛(r_{1},r_{2}, … r_{n}) ∋ r ≤ r_{i} ∀ i=//1,2,3// … n} and by **finiteness**, r<0
* claim that (x-r,x+r) ⊆  &{n}⋂_{i=1}(a_{i},b_{i})
* proof:
	* let y ∈ (x-r,x+r) ∋ x-r<y<x+r
	* → ∀ i = //1,2,3//,… n, a_{i}<x-r_i<x-r<y<x+r<x+r_{i}<b_{i}
	* i.e  y ∈(a_{i},b_{i})∀ i=//1,2,3,// … n} ∴ y ∈ (x-r,x+r) →  y ∈  ^{n}⋂_{i=1}(a_{i},b_{i})
	* * ∴ (x-r,x+r) ∈  ^{n}⋂_{i=1}(a_{i},b_{i})
