## Sign
Trả về dấu của một số.

### Các phiên bản
```glsl
float sign(float x)  
vec2 sign(vec2 x)  
vec3 sign(vec3 x)  
vec4 sign(vec4 x)
```

### Các tham số
```x``` số cần kiểm tra dấu âm dương.

### Mô tả
```sign()``` trả về -1.0 nếu x < 0.0, 0.0 nếu x = 0.0, và +1.0 nếu x > 0.0.
