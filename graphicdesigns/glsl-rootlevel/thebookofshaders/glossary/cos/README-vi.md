## Cos
Tính hàm cosine của 1 góc lượng giác

### Các phiên bản
```glsl
float cos(float angle)  
vec2 cos(vec2 angle)  
vec3 cos(vec3 angle)  
vec4 cos(vec4 angle)
```

### Các tham số
```angle``` góc lượng giác (đơn vị radian) cần tính cosine

### Mô tả
```cos()``` trả về giá trị cosine tương ứng với góc ```angle```
