## Bool
Kiểu dữ liệu boolean

### Các phiên bản
```glsl
bool aBool = true;
bool bBool = bool(aInt);
bool cBool = bool(aFloat);
```

### Mô tả
```bool``` chỉ nhận 1 trong 2 giá trị đúng hoặc sai
