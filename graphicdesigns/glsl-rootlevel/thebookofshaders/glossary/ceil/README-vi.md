## Ceil
Tìm số nguyên lớn hơn hoặc bằng gần nhất (làm tròn lên).

### Các phiên bản
```glsl
float ceil(float x)  
vec2 ceil(vec2 x)  
vec3 ceil(vec3 x)  
vec4 ceil(vec4 x)
```

### Các tham số
```x``` giá trị cần làm tròn

### Mô tả
```ceil()``` làm tròn ```x``` thành số lớn hơn hoặc bằng.
