## Not
Phép logic nghịch đảo.

### Các phiên bản
```glsl
bvec2 not(bvec2 x)  
bvec3 not(bvec3 x)  
bvec4 not(bvec4 x)
```

### Các tham số
```x``` giá trị boolean cần đảo ngược.

### Mô tả
```not()``` đảo ngược ```x```. Kết quả là một vector boolean mà thành phần thứ ```i``` là kết quả của phép logic ```!x[i]```.
