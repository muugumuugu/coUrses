## Acos
Tính góc lượng giác ứng với giá trị cosine

### Các phiên bản
```glsl
float acos(float x)  
vec2 acos(vec2 x)  
vec3 acos(vec3 x)  
vec4 acos(vec4 x)
```

### Các tham số
```x``` giá trị cosine cần tìm góc lượng giác tương ứng

### Mô tả
```acos()``` góc lượng giác (đơn vị radian) ứng với cosine bằng ```x```.
