## Floor
Tìm số nguyên gần nhất nhỏ hơn hoặc bằng (Làm tròn xuống).

### Các phiên bản
```glsl
float floor(float x)  
vec2 floor(vec2 x)  
vec3 floor(vec3 x)  
vec4 floor(vec4 x)
```

### Các tham số
```x``` số cần làm tròn xuống

### Mô tả
```floor()``` trả về số nguyên gần nhất nhỏ hơn hoặc bằng với ```x```.
