## Normalize
Chuẩn hóa vector.

### Các phiên bản
```glsl
float normalize(float x)  
vec2 normalize(vec2 x)  
vec3 normalize(vec3 x)  
vec4 normalize(vec4 x)
```

### Các tham số
```x``` Vector cần chuẩn hóa.

### Mô tả
```normalize()``` trả về một vector cùng chiều với ```x``` và có độ dài bằng 1.
