## Asin
Tính góc lượng giác ứng với giá trị sine

### Các phiên bản
```glsl
float asin(float x)  
vec2 asin(vec2 x)  
vec3 asin(vec3 x)  
vec4 asin(vec4 x)
```

### Các tham số
```x``` giá trị sine cần tìm góc lượng giác tương ứng

### Mô tả
```asin()``` trả về góc lượng giác (đơn vị radian) tương ứng với giá trị sine bằng ```x```.
