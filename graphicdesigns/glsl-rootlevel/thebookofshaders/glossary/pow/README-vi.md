## Pow
Lũy thừa.

### Các phiên bản
```glsl
float pow(float x, float y)  
vec2 pow(vec2 x, vec2 y)  
vec3 pow(vec3 x, vec3 y)  
vec4 pow(vec4 x, vec4 y)
```

### Các tham số
```x``` cơ số.

```y``` số mũ.

### Mô tả
```pow()``` trả về kết quả của phép toán ```x``` mũ ```y```
