## Length
Tính độ dài cuuar vector.

### Các phiên bản
```glsl
float length(float x)  
float length(vec2 x)  
float length(vec3 x)  
float length(vec4 x)
```

### Các tham số
```x``` vector cần tính độ dài

### Mô tả
```length()``` trả về độ dài của vector.
