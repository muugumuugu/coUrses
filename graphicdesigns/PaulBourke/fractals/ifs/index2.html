Some colour examples

Return to main page

There are a number of ways to "colour" these images. The first is to base the colour of each point on the iteration number. The first few images below have the first 100000 iterations drawn, the colour is varied from blue (zeroth iteration) to red (iteration 100000).

Another method is to use a fixed colour for each random choice of (a,b,c,d,e,f). In the next few images there are 4 choices for each (a,b,c,d,e,f), the following colours are assigned to each choice: green, yellow, blue, and magenta.
