ASKAP Site Visualisation

P.D. Bourke

Deep surveys of the Radio Universe with SKA Pathfinders Conference
March/April 2008


Abstract

I will present some recent work aimed at visualising the ASKAP site in a real-time interactive virtual environment. Every attempt is made to present the site as accurately as the available data permits while at the same time create a software package that runs at real-time rates on a range of hardware platforms and operating systems. The solution has been to leverage an existing game development engine, this provides both a powerful environment in which to deploy the various digital assets and results in a visually engaging experience. The current state of the software will be available for download at the conference.

Keywords

gaming, immersive environments, virtual reality, ASKAP, telescope dish, Boolardy
