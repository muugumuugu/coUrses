Case 12: 8 x 30inch tiled display

Return to index

The goal here was to determine how far one could go with a high resolution display driven from a single computer. The advantage of this was the ability to use/test a range of software packages. The disadvantage is twofild, firstly performance related to one machine handling 32MPixels. The other is the lack of compensation for the gap between the display regions, something that tiled display software would normally perform since it adds to the believability that it is a single display with occluding frames..
