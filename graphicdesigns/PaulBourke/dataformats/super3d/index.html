Super3D Text Export Format


I don't yet have a description of this format, if you can supply it then I'd like to hear from you. Each line of the file contains an optional line number followed by a semicolon, then a keyword followed by any data required by that keyword. The format essentially describes polygons with a colour (pen keyword) and normal (normal keyword).

So that you might recognise the format if you ever see it, here's an example of a cube 
0;      pen     0.0     0.0     0.0
1;      pen     0.0     0.0     0.0
normal  0       0       0

polygon 57.735  57.735  -57.735
