Mechanical Plotter Drawings (6)

I've been working on a way to draw wide lines on the plotter. Initially I wanted to be able to draw completely dense paths, but the resulting code is quite versatile. That means it is also possible to make "filled paths" as you can see in the architectural drawing below.

Drawing 0a1c88a

It also turned out to be a nice way to get PCB-like patterns, which I've been wanting to try for a long time. The below plots essentially use the same path offset method. However the initial path is repeatedly offset, while occasionally adding some noise.

Sold Drawing a5d0499
Sold Drawing 0ba65d8

The plots are drawn on Fabriano Bristol paper using Microns, and they are approximately A4 in size All the plots have some extra padding, so I suggest cutting them a little before framing them.
