CSS selectors

Name of tag


  p {
  color: red;
}

Classes


  .myclass {
  color: red;
}

ID's


  #myid {
  color: red;
}

Pseudo selectors


  p:first-child {
  color: red;
}

Can be used at the same time


  p.myclass {
  color: red;
}

Space means "inside"


  p .myclass {
  color: red;
}

There's no limit


  html body p strong.myclass {
  color: red;
}

CSS Properties

No one remembers them all

developer.mozilla.org

Styles the selection


  p {
  color: red;
}

Can change color, size, position, etc


  p {
  color: red;
  width: 200px;
}

CSS Values

Units


  p {
  width: 200px; /* width in pixels */
  width: 50%; /* width in percentage of parent */
  width: 4em; /* width in font size scale */
}

Colors


  p {
  background-color: red; /* color in name */
  background-color: #FF000; /* color in hex */
  background-color: rgb(255,0,0); /* color in RGB */
}

Other


  p {
  display: inline;
  border: 1px solid black;
}

Cascading Style Sheets

HTML as a tree


  <html>
  <body>
    <h1>This is <strong>heading</strong></h1>
    <p>This is <strong>some text</strong></p>
  </body>
</html>

Later styles overwrite earlier ones


  p {
  color: red;
  color: green;
}

... another example


  p {
  color: red;
}
p {
  color: green;
}

But the most specific always wins


  p.myclass {
  color: red;
}
p {
  color: green;
}
